HTML - це мова розмітки для створення веб-сторінок.
Вона використовує теги для структурування контенту.

CSS - це мова стилів, яка використовується для оформлення веб-сторінок.
Вона дозволяє змінювати кольори, шрифти та розташування елементів.

JS - це мова програмування, яка робить веб-сторінки інтерактивними.
Вона часто використовується для створення динамічного контенту.
